软件使用推荐-Mac

本文最后更新于：2026年2月10日 晚上

使用Mac Air两年，最近越发顺手，想写一篇文章记录我在mac上使用过的生产力软件，方便自己换机时下载，读者朋友们也可以借鉴和推荐。

以下工具基本覆盖了我在 开发、科研、写作、日常效率 中的高频使用场景。

AI的发展实在是过于迅速，短短一个月，我又体验了一些工具，有些甚至成为主力。— 2026.2.10

2026.2.10 新增

  • Trae(国际版): Trae周年庆，送了一个月600次的solo, 上手体验了一下，真的很不错。软件的更新频次过快，稳定性欠佳，我在使用期间提了两次bug issue

  • Antigravity: 仅尝试。

  • CodeX: 狙击opus 4.6. 我之前一直在VS code中用CodeX，有一天推送了CodeX发布APP的通知，就下载了看看。肯定是好用，但是我不知道和在VSCode中使用有什么区别。

  • Zotero: 回归经典。Zotero的插件社区太好了，AI集成，拓展性，可玩性都十分高。可以充分定制化，并且结合Zetero Daily 每天向我的邮箱中推荐10篇论文，真的是很香。所有的功能都可以集成AI，在Google API强大模型的加持下，让我拥有十分衬手的文献管理工具。

  • CleanMyMac: mac 清理工具，可惜是付费软件。正在探索好用的开源

Homebrew：包管理工具，Mac必装软件

开发 IDE / 编辑器

  • IntelliJ IDEA：Java 主力 IDE，陪伴了我的所有java项目
  • Cursor：不多说，最强AI代码编辑器。
  • Visual Studio Code：神

终端 / 远程连接

  • iTerm2：终端主力，替换macos中的原生终端使用f

  • Termius：服务器连接工具，替代xshell

  • Apifox：接口文档 + 调试 + Mock 替换postman，UI设计好，同时可以直接做压测。

写作 / 文档

  • Typora：Markdown 写作-付费软件。
  • ONLYOFFICE：开源PDF文档阅读器，替代Adobe。
  • Class Verge：代理软件。
  • 小绿鲸：读论文、标注、整理（偏文献阅读工作流）。
  • Easydict：查词/翻译。
